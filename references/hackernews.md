# Hacker News 抓取参考

## 目标

用 Hacker News 捕捉偏工程、产品、创业和独立开发者视角的 AI 趋势，重点判断企业采用的技术可行性、成本、维护负担和产品成熟度。

## 优先关注

- Show HN
- Ask HN
- 带有 `AI`、`LLM`、`agent`、`Claude`、`OpenAI`、`MCP`、`RAG`、`enterprise`、`copilot`、`workflow` 的帖子
- 官方发布或工程博客在 HN 上引发的讨论
- 独立项目、开发者工具、agent 工具、MCP 工具、AI coding 工具

## 搜索关键词

- `enterprise AI`
- `AI adoption`
- `AI governance`
- `LLM`
- `agent`
- `MCP`
- `RAG`
- `Claude`
- `OpenAI`
- `Copilot`
- `developer productivity`
- `workflow automation`
- `automotive AI`

## 抓取字段

- 标题
- 分数
- 评论数
- 发布时间
- 原文 URL
- HN 讨论 URL
- 讨论里的核心观点
- 初步分类：基础研究 / 企业应用 / 创业前沿 / 风险信号
- 对企业 adoption 或汽车研发中心的潜在影响

## 保留标准

- 讨论不是单纯站队
- 评论区出现工程判断、成本判断、维护判断、供应商锁定判断、安全判断或产品成熟度判断
- 能反映“技术圈正在认真看什么”以及“为什么企业采用可能困难”
- Show HN 项目只有在代表未来企业工作流、工具链或创业方向时保留

## 观察重点

- 独立项目是否在形成可迁移到企业的工具形态
- 工程师对 agent、MCP、RAG、AI coding 等方向的真实态度
- 评论里提到的限制、维护成本、部署复杂度和落地难点
- 哪些 Show HN 值得进入后续观察池或供应商监控名单
- 是否出现与企业安全、on-prem、私有化、审计、数据治理相关的讨论
